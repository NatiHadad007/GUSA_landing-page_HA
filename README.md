# GUSA_landing-page_HA
A responsive sign-up module, including a landing page top section and a pop-up modal.

- **Landing Section:** Introductory text, imagery, and an **"Apply Now"** button.  
- **Sign-Up Modal:** Appears when clicking the **"Sign Up"** button.  

Built with **HTML, CSS, and JavaScript**.
